# Forge & Fable: Arcane Observatory Matte Revision

This is a self-contained browser prototype of the revised full-site art direction.

## Revision goals

- Remove glassmorphism, blur, translucent panels and neon glow
- Replace star fields and perspective constellations with flat engraved astronomical instruments
- Make the application shell feel built from painted wood, slate, iron and brass
- Keep parchment only for documents, notes and handouts
- Preserve the existing interactive prototype pages and behavior

Open `index.html` in a desktop browser. Use the black prototype navigation bar to switch pages.
