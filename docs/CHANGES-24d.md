# Changes 24d — The Record

## Delivered

- The Record merges campaign rolls and table events in chronological order.
- Filter controls show all entries, rolls only, or table events only.
- Announcements, rests, conditions, handouts, cues, and roll requests receive readable register text.

## Limits

- The record is backed by the existing bounded campaign roll/event histories (200 records each), not an unbounded audit archive.
- Starting/stopping looping music is stateful and is not currently emitted as a separate record line.
