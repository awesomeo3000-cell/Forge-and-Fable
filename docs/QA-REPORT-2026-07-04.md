### 8. Dice Roller Agent ✅ GREEN

#### Dice Types & Rendering
- All 7 polyhedrals: d4, d6, d8, d10, d12, d20, d100
- d20: True 3D icosahedron — 20 triangular faces with CSS `matrix3d`, golden-ratio vertex geometry, per-face brightness from "studio light" direction
- d4–d12, d100: Procedural polyhedral meshes (tetrahedron, cube, octahedron, trapezohedron, dodecahedron)
- SVG fallbacks exist for all dice

#### Roll Animation
- `ROLL_ANIMATION_MS = 2800`, `DEFAULT_RESULT_LINGER_MS = 1800`
- Internal 3D tumble: 1500ms, 20 keyframes, easeOutBack
- Die staggering: 220ms (simple) / 180ms (pools/d20)
- Crit (NAT 20): "HOLY SHIT" banner at 1680ms; Fumble (NAT 1): "FUMBLE" banner
- Respects `prefers-reduced-motion`: snaps to rest position immediately ✅

#### Roll Logic
- `rollDie(sides)`: `crypto.getRandomValues()` with rejection sampling; uniform [1, sides] ✅
- `scoreFrom4d6()`: 4d6 drop lowest, sums top 3 ✅
- Advantage: 2d20, keep higher; Disadvantage: 2d20, keep lower
- Mode is one-shot — resets to "normal" after each roll
- Armor penalty forces disadvantage on STR/DEX ability rolls

#### Roll Integration
| Trigger | How |
|---------|-----|
| Ability check | `rollD20ForAbility(name, mod+effChecks, key)` |
| Saving throw | `rollD20ForAbility(name, saveBonus, key)` |
| Skill check | `rollD20ForAbility(skill.name, bonus, skill.ability)` |
| Initiative | `rollD20ForAbility("Initiative", initiative, "dexterity")` |
| Attack roll | Row click → `rollD20ForAbility(name, toHit, ability)` |
| Weapon damage | Small buttons → `props.onRoll(name, sides, count, mod)` |
| Spell attack | `rollD20(name attack, spellAttack)` |
| Spell damage | `props.onRoll(name, sides, count, 0)` from `parseDamageDice()` |
