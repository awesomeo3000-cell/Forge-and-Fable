# Changes 24e — Handouts and player table strip

## Delivered

- The DM can share a titled image URL as a handout event.
- Players receive a table notification and a dismissible lightbox for the handout.
- While a player is in a campaign, a docked Table strip displays the round, visible current combatant, latest announcement, audio controls, and an Open campaign action.

## Verification

- Build and automated tests verify the component and API paths compile and pass.
- Manual two-chair browser verification remains required for the handout-to-lightbox roundtrip and visual turn-state review.
