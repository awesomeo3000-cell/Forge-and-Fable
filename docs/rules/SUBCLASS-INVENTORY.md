# Subclass Inventory

The source inventory is stored in `rules-research/subclasses/inventory.json`.

- 2014 coverage: 28 SRD subclass records plus four licensed Tasha’s Artificer specialists.
- 2024 coverage: the twelve official Basic Rules subclass records, one for each core class.
- 2024 licensed coverage beyond the Basic Rules records remains pending and is not exposed in production.
- Every record has an edition-specific ID, base-class ID, source ID, feature-level schedule, and coverage status.

Subclass feature records are the next research layer. Inventory completion does not mean the feature text, choices, spell lists, or resource interactions have been production-approved.
