# Subclass Inventory

The source inventory is stored in `rules-research/subclasses/inventory.json`.

- 2014 coverage: 28 SRD subclass records plus four licensed Tasha’s Artificer specialists.
- 2024 coverage: the twelve official Basic Rules subclass records, one for each core class.
- 2024 licensed coverage beyond the Basic Rules records remains pending and is not exposed in production.
- Every record has an edition-specific ID, base-class ID, source ID, feature-level schedule, and coverage status.

The detailed 2014 Basic Rules feature packet is stored at `rules-research/subclasses/2014/basic-rules.json` and currently covers the first four comparison subclasses. The detailed 2024 Basic Rules feature packet is stored at `rules-research/subclasses/2024/basic-rules.json` and covers all twelve 2024 core subclasses. Both use structured feature levels, choices, spell changes, resource interactions, parent-feature dependencies, and source references. These records are research-only until the independent second review and progression-engine integration are complete.

2014 subclass feature extraction and additional licensed 2024 coverage remain pending.
