# Cross-System Research Inventory

The following systems must be edition-specific where their timing, formula, or behavior differs:

- Ability Score Improvements, feats, Origin feats, Fighting Style feats, and Epic Boons.
- Full, half, third, and pact spellcasting.
- Cantrips, known spells, prepared spells, spellbooks, replacement, and always-prepared spells.
- Expertise, fighting styles, Weapon Mastery, invocations, Metamagic, and maneuvers.
- Rage, Bardic Inspiration, Channel Divinity, Wild Shape, Second Wind, Action Surge, Indomitable, Focus/Ki, Lay on Hands, Sorcery Points, Arcane Recovery, and Sneak Attack.

Each packet must document initial value, scaling, recharge, UI representation, level-up choice behavior, and server validation.
