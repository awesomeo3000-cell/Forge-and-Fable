# Forge & Fable Rules Source Ledger

This ledger records the source precedence and edition boundaries for class and subclass research.

## Precedence

1. Latest official errata.
2. Latest official digital compendium text.
3. Latest official printing.
4. Original sourcebook text.
5. Official clarifications that explicitly change an interpretation.
6. Secondary references for discovery only.

## Sources

| ID | Ruleset | Title | Official source | Status | Notes |
|---|---|---|---|---|---|
| `2014-srd-5.1` | 2014 | System Reference Document 5.1 | [Wizards SRD resources](https://dnd.wizards.com/resources/systems-reference-document) | active | Current production rules baseline; existing class data is being rechecked against this source. |
| `2024-basic-rules` | 2024 | D&D Beyond Basic Rules: Character Classes | [D&D Beyond](https://www.dndbeyond.com/sources/dnd/br-2024/character-classes) | research | Official digital source used for the Barbarian pilot and the strict-2024 baseline. |
| `2014-tce-artificer` | 2014 | Tasha's Cauldron of Everything: Artificer | licensed source catalog | research | Existing production Artificer is treated as a 2014-compatible Tasha source, not as an assumed 2024 class. |

## Research rules

- Research records remain separate from production data until approved.
- A source ID is required for every class, subclass, feature, resource, and spell grant.
- Same-named options from different rulesets receive separate IDs.
- A source conflict is recorded in `ERRATA-LOG.md`; it is never resolved by silently merging text.
