## What changed

1. **R18 tokens** appended to `globals.css` under the banner comment, exactly as specced in §1 (`--ledger-paper` … `--ledger-seal`, plus a `--ledger-font` alias for `var(--font-newsreader), Georgia, serif`). No existing block edited.
2. **Masthead.** Deleted the `Sparkles` brand glyph and the `Character Studio` strong line. New brand block is two stacked lines — `FORGE & FABLE` (11px, 0.22em letterspaced caps, `--parchment-2`) over `Character Ledger` (18px Newsreader, `--parchment`) — via `ledger-masthead` added *beside* `builder-brand` (old class retained). Top bar (`ledger-topbar` beside `builder-topbar`) is now flat `--ground-2` with `border-bottom: 1px solid var(--rule-soft)`, `backdrop-filter: none`, no radius.
3. **Top-bar actions.** Campaigns/feedback/logout buttons got `ink-action` beside `glass-icon`: borderless, backgroundless, `--parchment-2` at rest → `--parchment` on hover; kept their 18px lucide icons (§0.7 exception). `account-chip` got `ledger-account`: plain letterspaced small-caps text, no chip background/border; the `UserRound` icon was removed (not a top-bar *action button*, so no icon exception applies).
4. **Roster rail.** `vault-rail` gained `ledger-rail`: `--ledger-paper` background + the standard `--grain` overlay (copied from the `.paper-surface::before` pattern, not reinvented), `--ledger-edge` border, `--r-sm` radius, `--shadow-soft`. Heading is now `ROSTER` in 11px letterspaced caps `--ledger-faded` with a hairline below, and the count marginalia (*two souls recorded* — spelled out one–nine, numerals from 10; `soulsRecorded()` in `ledgerCopy.ts`) in italic `--ledger-ghost`.
5. **Import/New buttons** became quiet ink text buttons (`ink-action-dark` beside `glass-icon`): small-caps `IMPORT` / `NEW`, `--ledger-faded` → `--ledger-ink` on hover. **Deviation from spec, flagged:** the proposal keeps these as buttons but §0.7 bans lucide icons on this surface outside the top bar and modal closes, so the `Upload`/`Plus` glyphs were replaced with text labels rather than keeping banned icons or shipping empty buttons. `title` attrs kept.
6. **Roster rows.** `vault-avatar` gained `ledger-row`: full-width ruled rows (hairline `border-bottom`, no per-row box/radius/gap — `vault-list` gap zeroed), 26px initial-circle (first letter of the name, `background: var(--class-a)` via the existing `data-class` attr, text `--ledger-paper`), name 15px `--ledger-ink`, under it one italic 12px `--ledger-faded` line `{Species} · {Class} · {ordinal} level` (`ordinalLevel()` handles 1st/2nd/3rd/11th/21st). `ClassIconPlaceholder` removed from the row (import dropped from this file; the class SVGs still live in the builder and sheet). Added `aria-pressed` to rows (previously the active row had no announced state — §0.4 says a11y must not regress; this is an addition, not a removal).
7. **Active/hover states.** Active: `--ledger-tint` background + 3px `--ledger-seal` left rule, nothing else. Inactive hover: `color-mix(in srgb, var(--ledger-tint) 55%, var(--ledger-paper))`. Transform/shadow explicitly zeroed.
8. **Ghost row** after the last character: `✦  Inscribe a new name…` (13px italic `--ledger-ghost`), same handler as the New button. With an empty roster it reads `✦  Inscribe a new name in the roster…` and the count marginalia is suppressed.
9. **New module `src/lib/ledgerCopy.ts`** — the §2 copy deck: `CLASS_DESCRIPTORS` (+ `classDescriptor()` fallback-"" helper), `BUILD_MODE_DESCRIPTORS`, `CHAPTERS` (numerals, names, subtitles, per-chapter button labels), `soulsRecorded()`, `ordinalLevel()`, `firstSentence()` for later sub-rounds.
